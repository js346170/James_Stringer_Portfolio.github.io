There is a strong emphasis on cross curiculur learning as statistics and project design were a major focus. This was also my first expereience with coding as I learned to use Statistical Analysis Systems.

  • Qualitative and quantitative data analysis
  • Fieldwork techniques and participant observation
  • Cross-cultural communication and understanding
  • Ethical considerations in research
  • Data visualization and presentation skills
  • Statistical analysis using software like SAS and SPSS
