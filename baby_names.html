Baby Names Data Analysis

This project analyzes trends in baby names over time using a comprehensive dataset. The analysis includes:

  • Popularity trends for specific names (like "Joseph" and "Luna")
  • Diversity metrics including Shannon Entropy and Herfindahl-Hirschman Index
  • Statistical analysis of name diversity changes over time
  • Visualizations of naming patterns by first letter and gender

The project demonstrates skills in data cleaning, statistical analysis, and visualization using Python, Pandas, and Matplotlib.

View on GitHub View on Google Drive

Key Findings

The analysis revealed several interesting patterns:

  • A significant increase in name diversity over time (confirmed by statistical tests)
  • Changing popularity of names starting with different letters
  • Gender differences in naming trends
  • The decline in dominance of traditional top names
